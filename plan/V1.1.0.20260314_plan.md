# V1.1.0 (2026-03-14) 调整计划与变更记录

## 目标
根据需求完成以下四项调整：
1. 添加记录 -> 日期中日历界面英文改为简体中文适配
2. 项目所有导入/导出提示改为简体中文
3. 使用根目录 ico 图标
4. 软件名统一为“短险卡记录”

## 变更明细

### 1) 日期选择器简体中文适配
- 文件：`pubspec.yaml`
  - 新增依赖：`flutter_localizations`（SDK 内置）
- 文件：`lib/app/app.dart`
  - 新增：
    - `localizationsDelegates`
    - `supportedLocales: [Locale('zh', 'CN')]`
    - `locale: Locale('zh', 'CN')`
- 文件：`lib/features/records/record_form_page.dart`
  - `showDatePicker` 增加：`locale: const Locale('zh', 'CN')`

### 2) 导入导出提示简体中文化
- 文件：`lib/features/settings/settings_page.dart`
- 已调整提示：
  - 导出成功提示：`导出成功，文件已保存到：...`
  - 导入成功提示：`成功导入 x 条记录`
  - 导入失败提示：`导入失败，请检查 JSON 文件格式是否正确`
  - 未选择文件提示：`未选择文件`

### 3) ico 图标（根目录）接入
- 图标文件：`/favicon.ico`
- 文件：`windows/runner/Runner.rc`
  - 应用图标资源由
    - `resources\\app_icon.ico`
  - 改为
    - `..\\..\\favicon.ico`

> 说明：`.ico` 主要用于 Windows 图标资源；Android APK 正常流程需使用 PNG（mipmap）图标。

### 4) 软件名统一为“短险卡记录”
- 文件：`lib/app/app.dart`
  - `MaterialApp.title = 短险卡记录`
- 文件：`android/app/src/main/AndroidManifest.xml`
  - `android:label = 短险卡记录`
- 文件：`ios/Runner/Info.plist`
  - `CFBundleDisplayName = 短险卡记录`
  - `CFBundleName = 短险卡记录`
- 文件：`windows/runner/Runner.rc`
  - `FileDescription/InternalName/ProductName` 等产品名字段同步为“短险卡记录”

## 版本更新
- 文件：`pubspec.yaml`
- 版本从 `1.0.0+1` 调整为 `1.1.0+2`

## 验收建议
1. 运行应用后打开“新建记录”，验证日期选择器为中文界面
2. 在设置页执行导入/导出，确认提示文案均为简体中文
3. Windows 构建后确认应用图标使用根目录 `favicon.ico`
4. Android/iOS 启动后确认应用显示名为“短险卡记录”
